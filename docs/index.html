Operative Note Generator

A focused operative-note drafting tool for common emergency general surgery cases. Structured inputs in, reviewable note out — without accounts, storage, analytics, or a backend.

Open generator View source

What it is

A static web app that converts clinician-entered operative details into a structured draft operation note for copying into the appropriate clinical record after review.

Why it exists

Repetitive operation notes are easy to make inconsistent under pressure. This keeps the common fields, procedure-specific prompts, checks, and output format close together.

Current status

Personal prototype, actively iterated, and deliberately narrow. It supports a small set of common general surgical procedures rather than trying to be a universal theatre system.

Current prototype

Structured prompts for common emergency general surgery notes

  • Procedure selector for lap appendix, lap chole, diagnostic laparoscopy, abscess I&D, and open inguinal hernia repair.
  • Shared admin, anaesthetic, indication, findings, specimen, drain, closure, and post-operative plan fields.
  • Procedure-specific fields and advisory checks to reduce obvious omissions before copying the note.
  • Generated text remains editable by review: the tool does not infer pathology, complications, or clinical decisions.
  • Runs entirely in the browser with no backend, login, cloud sync, analytics, external AI calls, or data transmission.

Clinical safety boundary

Drafting aid only. Not a clinical system.

This is a personal static tool for drafting text from information entered by a clinician. It is not an EHR, referral platform, NHS-approved product, clinical safety system, regulated medical device, or source of truth.

No patient-identifiable information should be sent to this website or repository. Generated notes must be checked, corrected, and entered into the appropriate clinical record by the responsible clinician.

Operative Note Generator is an experimental personal project by Sangeev. It runs as static files and should be treated as clinician-reviewed drafting assistance only.
