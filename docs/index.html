Op notes / Scratchpad

Operative note generator

I made this because I was tired of retyping the same structure after common emergency general-surgery cases.

It is just a browser form that turns fields into a draft operation note. No login, backend, analytics or autosave.

Use it as a starting point, then check the note yourself.

Open generator Source on GitHub

Do not enter patient-identifiable information.

This tool runs entirely in your browser. There is no login, backend, analytics or autosave.

Review the generated text carefully before using it in any clinical record.

What it does

It turns structured fields into a plain-text draft operation note that can be checked and edited before copying into the record.

Currently covered

  • laparoscopic appendicectomy
  • laparoscopic cholecystectomy
  • diagnostic laparoscopy +/- washout / adhesiolysis
  • incision and drainage of abscess
  • open inguinal hernia repair
  • open umbilical hernia repair
  • emergency laparotomy

Example appendicectomy draft

Synthetic details. The tool does not store, transmit or check the clinical truth of anything entered.

Limits

It does not store data, make clinical decisions, validate completeness or replace local documentation standards.

Status

Personal tool. Static site. No external services required to draft a note in the browser.
